Linux PATH环境变量以及常用快捷键

首先是一些常用的命令:

  1. mkdir: 新建一个文件

  2. ip …: 与网络有关的一系列命令

  3. whereis + 命令: 查找命令文件所在的位置

  4. echo $PATH: 查看当前的环境变量所在的文件

  5. cp: 复制一个文件（第一个参数）到另一个文件的位置（第二个参数）

  6. export: 新增、删除、修改环境变量（对变量的修改只在本次终端中有效）

参数:

  • -f 　代表[变量名称]中为函数名称
  • -n 　删除指定的变量。变量实际上并未删除，只是不会输出到后续指令的执行环境中
  • -p 　列出所有的shell赋予程序的环境变量

方向键: 查找历史命令

设置环境变量: 将一个新的与环境变量相关的文件添加到环境变量相关的文件中

export PATH$PATH: ＋文件的位置

上面的修改只是暂时的，要想永久修改环境变量，需要修改/etc/profile文件

通过路径使用命令:

  1. 绝对路径: 绝对路径＋文件名，如 /home/gokej/work/my_date

  2. 相对路径:

    • 当前位置的相对路径＋文件名，如 ./work/my_date
    • ..代表上一级目录＋文件名，如 ../gokej/work/my_date

    我们可以发现 . 代表当前目录，..代表上一级目录
