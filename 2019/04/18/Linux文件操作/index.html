Linux文件操作

本文涉及到的文件操作有创建、查看、写入、复制、删除、重命名。

一些常用的命令：

mkdir：即make directory，语法为　mkdir [options] dirname，可以在当前目录下或在指定目录下创建一个目录，即文件夹

touch：语法为 touch [options] filename，可在当前目录下火灾指定目录下创建一个文件

vim：语法为 vim filename，用vim打开一个可读的文件。

注：要想查看文件的二进制内容，可用vim输入:%!xxd命令

cp：即copy，语法为 cp filename1 filename2（或其他），即在当前目录下将file1复制到file2

rm: 即remove，语法为rm 参数 filename，即将文件删除

可选的参数：

  • -f　强制删除文件
  • -r　递归删除文件
  • -i　确定后再删除文件
  • -rf　删除当前目录下的索引文件且不可恢复（著名命令（滑稽））

file：语法为file filename，查看文件的类型

tar：语法为tar filename1 filename2，生产file2文件的压缩文件，名为file1

mv：语法为mv filename1 filename2，将file2文件名修改为file1

值得注意的是，Linux下与windows不同，文件的类型与文件后缀名没有关系，只与其内容有关
